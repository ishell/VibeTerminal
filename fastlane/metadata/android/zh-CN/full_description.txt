VibeTerminal 是一款功能强大的 Android SSH 终端客户端：

功能特点：
• 完整的终端模拟，支持 ANSI/xterm
• Zellij 多路复用器集成，支持面板管理
• VS Code 风格的迷你地图，快速导航
• Path 风格浮动键盘，交错动画效果
• Termius 风格键盘栏，常用快捷键
• 基于项目的连接管理
• Claude Code 会话对话历史追踪
• SQLCipher 加密数据库，安全存储凭证
• Nerd Font 图标支持
• 深色主题优化

安全特性：
• SSH 密钥认证（RSA、Ed25519、ECDSA）
• 使用 Android Keystore 加密存储凭证
• 主机密钥指纹验证

本应用专为需要在 Android 上获得强大终端体验的开发者设计。
